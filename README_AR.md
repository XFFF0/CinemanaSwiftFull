# CinemanaSwiftFull

مشروع SwiftUI كامل مع:
- Home حقيقي من API
- Banner
- Latest
- Search
- Details
- Qualities / 4K
- Subtitles
- Player
- Download
- Account
- GitHub Actions build.yml
- XcodeGen project.yml

## البناء على GitHub Actions
1. ارفع كل الملفات إلى الريبو
2. شغل Actions
3. بعد النجاح حمل Artifact باسم CinemanaSwiftFull-IPA

## ملاحظات
- المشروع يستخدم endpoints التي استخرجتها أنت.
- إذا تغيّر شكل JSON في endpoint معين، عدّل Model المقابل.
